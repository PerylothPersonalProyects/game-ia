# Publicación para LinkedIn

**LABORATORIO IA: Transformando el Desarrollo de Juegos con Agentes de IA Locales**

---

✨ **¿Qué construimos?**

Un juego idle clicker completo con características multiplayer, desarrollado utilizando agentes de IA locales (Ollama + OpenCode) que trabajan 100% offline, sin costos de API externas.

🛠️ **Stack Tecnológico:**

- **Frontend:** React 19 + TypeScript + Phaser 3 + Vite
- **Backend:** Node.js + Express + Socket.io + MongoDB
- **IA:** OpenCode conectado a Ollama (modelos locales: Llama3, DeepSeek-Coder, Mistral)
- **QA:** Karate Framework + Playwright

🎮 **Características del Juego:**

- Sistema de clicks con ingresos pasivos
- Compras de upgrades (Cursor, Abuela, Granja, Mina, Fábrica)
- Modo multiplayer con salas y tiendas de intercambio
- Persistencia en MongoDB
