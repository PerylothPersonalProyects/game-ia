# Catálogo de Upgrades

Este documento contiene las 100 mejoras disponibles en el juego.

## Formato de cada Upgrade

| Campo | Descripción |
|-------|-------------|
| `id` | Identificador único |
| `name` | Nombre de la mejora |
| `description` | Descripción que ve el jugador |
| `baseCost` | Costo inicial |
| `costMultiplier` | Multiplicador de costo por nivel |
| `effect` | Efecto (monedas por click o por segundo) |
| `maxLevel` | Nivel máximo |
| `type` | `click` o `passive` |
| `tier` | Tier (1-10) para raridad |

## Sistema de Progresión

### Costo
```
cost = baseCost * (costMultiplier ^ purchased)
```

### Efecto Total por Tipo
- **Click**: 1 (base) + Σ(effect * level) de todos los upgrades click
- **Passive**: Σ(effect * level) de todos los upgrades passive

### Distribución en el Shop
- Al comenzar: 4 upgrades aleatorios
- Al comprar: se reemplaza con uno nuevo
- Al hacer swap: se reemplaza el elegido
- Al hacer refresh: todos se regeneran
