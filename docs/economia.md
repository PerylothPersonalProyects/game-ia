# Economía del Juego

## Recurso Principal

**Coins** - La moneda del juego

### Generación

| Método | Descripción |
|--------|-------------|
| Click | Coins por interacción manual |
| Idle | Coins generados por segundo (passive income) |

---

## Upgrades

Cada upgrade tiene:
- **Costo base** - Precio inicial
- **Multiplicador** - Cómo crece el precio
- **Efecto** - Qué mejora (click o idle)

### Fórmula de Costo

```
costo = costoBase * (multiplicador ^ nivel)
```
