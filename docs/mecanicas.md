# Mecánicas del Juego

## Loop Principal

1. El jugador **hace click** para obtener coins
2. Los coins se usan para **comprar upgrades**
3. Los upgrades aumentan la **generación pasiva** de coins
4. Con más coins, el jugador puede comprar **mejores upgrades**
5. El ciclo se repite con dificultad/escala creciente

---

## Sistemas

### Click System
- Cada click genera una cantidad base de coins
- Puede mejorarse con upgrades

### Idle Generation System
- Genera coins automáticamente cada segundo
- Se incrementa con upgrades

### Upgrade System
- Compras que mejoran las estadísticas del jugador
- Cada upgrade tiene un costo que escala exponencialmente

| Nivel | Meta |
